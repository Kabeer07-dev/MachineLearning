# 🤖 Machine Learning Projects

A collection of machine learning and exploratory data analysis (EDA) projects built using Python and Jupyter Notebooks. This repository covers real-world datasets with hands-on analysis and predictive modeling.

---

## 📁 Repository Structure

```
MachineLearning/
│
├── heart.ipynb            # Heart Disease Prediction (ML Model)
├── heart.csv              # Dataset for heart disease analysis
│
├── EDA_insurance.ipynb    # Exploratory Data Analysis on Insurance Data
├── insurance.csv          # Dataset for insurance EDA
│
└── README.md
```

---

## 📓 Projects

### 1. 🫀 Heart Disease Prediction — `heart.ipynb`

A machine learning project that uses clinical features to predict the likelihood of heart disease in a patient.

**Dataset:** `heart.csv`

**Key Steps:**
- Data loading and preprocessing
- Exploratory Data Analysis (EDA) with visualizations
- Feature selection and engineering
- Model training and evaluation (classification algorithms)
- Performance metrics: accuracy, confusion matrix, etc.

---

### 2. 📊 Insurance EDA — `EDA_insurance.ipynb`

An in-depth exploratory data analysis of an insurance dataset to uncover patterns and relationships between variables such as age, BMI, smoking status, and insurance charges.

**Dataset:** `insurance.csv`

**Key Steps:**
- Data cleaning and inspection
- Statistical summaries
- Univariate and bivariate analysis
- Visualization of key trends and correlations
- Insights and observations

## 🚀 Getting Started

### Prerequisites

Make sure you have Python 3.x and the following packages installed:

```bash
pip install pandas numpy matplotlib seaborn scikit-learn jupyter
```

### Run the Notebooks

```bash
# Clone the repository
git clone https://github.com/Kabeer0700-gif/MachineLearning.git

# Navigate to the project directory
cd MachineLearning

# Launch Jupyter Notebook
jupyter notebook
```

Then open any `.ipynb` file from the Jupyter interface in your browser.
